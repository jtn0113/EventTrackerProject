# EventTrackerProject

## Description
This application is a tracker of the real life locations from your favorite movies and TV shows.

CRUD Operations:
- CREATE (POST)
  - 8084/api/locations
- READ (GET)
  - 8084/api/locations
  - 8084/api/locations/1
- UPDATE (POST)
  - 8084/api/locations/1
- DELETE (DEL)
  - 8084/api/locations/1
